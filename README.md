TS3Plugin-sys
=============
The documentation can be found [here](https://flakebi.github.io/rust-ts3plugin/doc/ts3plugin_sys/).

Usage
-----
Add the following to your `Cargo.toml`:
```
[dependencies]
ts3plugin-sys = "0.2"
```
