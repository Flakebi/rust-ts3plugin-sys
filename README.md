TS3Plugin-sys
=============

Usage
-----
Add the following to you `Cargo.toml`:
```
[dependencies.ts3plugin-sys]
git = "https://github.com/Flakebi/rust-ts3plugin-sys"
```

Dependencies
------------
The dependencies are resolved by cargo.

 - libc
 - bitflags
